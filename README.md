# TaskSharingSQliteData

This sample project demonstrates a setup for managing shared user groups, shared tasks, and private user-specific task data using SQLiteData in Swift.

## Steps to Reproduce

1.	On Device A, create a new Group A.
2.	Share Group A with Device B.
3.	On Device A, create a new Task A and assign it to Group A.
4.	Wait until Task A syncs to Device B.
5.	On Device B, attempt to complete Task A.
